Detailed Description

A QObject filter that installs a minimal context menu with an action to trigger a link statistics request.

Creates a QMenu with a minimal set of actions.

The following actions are created by the CuContextMenuConstructor:

  • A Link statistics action, to request information about the underlying link.
Parameters
parentthe parent widget of this menu.
parent_as_cwithe same object as parent must be passed. It must implement CuContextI interface.
Note.
Since it is not possible to cast CuContextI to QWidget and vice versa, it is necessary to store the parent reference both as QWidget and CuContextI. The linkStatsTriggered signal will contain the CuContextI pointer that will be used by the receiver to get the CuContext reference to access the cumbia-qtcontrols widget context.
